# YoudaoNoteLM V1.0 产品需求文档 (PRD)

> 版本：V1.0
> 日期：2026-06-03
> 状态：草稿

## 1. 执行摘要

### 1.1 问题陈述

学生、研究者和知识工作者在学习和研究过程中面临三个核心痛点：

1. **资料分散**：笔记、PDF、网页、音频等多种格式的资料散落各处，难以统一管理和检索。
2. **被动阅读**：现有工具仅提供存储和阅读能力，无法主动将资料转化为结构化的学习资产（思维导图、测验等）。
3. **知识断层**：从"收集资料"到"真正理解"之间缺乏有效的转化路径，知识停留在表面。

### 1.2 解决方案

YoudaoNoteLM 是一款以**笔记本**为核心的个人知识管理与学习助手，对标 Google NotebookLM。用户可导入多种格式的资料，基于资料内容与 AI 对话，并一键生成思维导图、测验、PPT 和笔记，实现从**收集 → 理解 → 内化**的学习闭环。

核心差异化：**多 Agent 协作架构**——系统由一个主 Agent 统一调度，下辖检索 Agent、对话 Agent 和生成 Agent 三个专职 Agent。主 Agent 负责理解用户意图、拆解任务并分派给合适的 Agent 执行：检索 Agent 从资料中语义检索相关内容，对话 Agent 基于检索结果生成自然语言回复，生成 Agent 负责产出思维导图、测验、PPT 和笔记等内容。用户既可通过自然语言对话触发生成，也可通过 UI 按钮一键生成，两条路径共享同一套 Agent 调度逻辑，结果统一汇入笔记区域。


## 2. 用户体验与功能

### 2.1 用户画像

| 角色 | 场景 | 核心需求 |
|------|------|----------|
| **学生** | 复习备考、写论文 | 将课件/笔记导入，自动生成测验检验理解程度 |
| **研究者** | 文献综述、课题研究 | 批量导入论文 PDF，AI 辅助提炼关键观点 |
| **知识工作者** | 会议记录、培训资料整理 | 导入音频/文档，快速生成结构化笔记和思维导图 |
| **有道云笔记用户** | 已有大量有道云笔记，希望统一管理 | 授权后一键导入已有笔记，基于笔记内容与 AI 对话 |

### 2.2 用户故事与验收标准

#### 模块一：资料导入

**US-1.1 文件导入**
> 作为用户，我希望导入 PDF/DOCX/TXT/MD/ 格式的文件，以便将本地资料统一管理。

验收标准：
- 支持 `.txt`、`.md`、`.docx`、`.pdf`、`.pptx` 等格式
- 单文件大小限制：文档类 ≤ 30MB，音频类 ≤ 300MB
- 上传后自动解析为 Markdown 格式并可以切换显示解析结果和原格式内容
- 上传中展示加载状态
- 导入失败时显示明确的错误原因

**US-1.2 网址导入**
> 作为用户，我希望粘贴一个网页 URL，系统自动抓取正文内容并导入。

验收标准：
- 支持 HTTP/HTTPS 协议的网页
- 自动提取正文，去除导航栏、广告等干扰内容
- 提取结果以 Markdown 格式存储
- 仅支持单页抓取，不递归子页面
- 不支持需要登录的页面（V1 限制）

**US-1.3 音频导入**
> 作为用户，我希望上传音频文件，系统自动转写为文本。

验收标准：
- 支持 `.mp3`、`.wav` 格式
- 使用语音识别服务转写为文本
- 转写结果转为 Markdown 格式
- 仅支持中文音频（V1 限制）

**US-1.4 有道云笔记授权与导入** （**暂定**）
> 作为用户，我希望授权有道云笔记账号后，直接在应用内浏览我的有道云笔记内容并一键导入为资料来源。

验收标准：

**授权流程**：
- 导入入口提供"绑定有道云笔记"按钮
- 点击后弹出引导弹窗，分步骤说明如何获取 API Key（网易智能开发者平台）
- 用户粘贴 API Key 完成授权绑定
- 授权成功后状态持久化，后续访问无需重复授权
- API Key 加密存储，关联到用户账号，不泄露到日志
- 提供"解除绑定"功能，清除已存储的 API Key

**笔记浏览**：
- 授权成功后，展示有道云笔记的目录/笔记列表（支持分页、目录过滤）
- 支持按标题搜索笔记
- 点击笔记可预览其内容（Markdown 格式）
- 列表显示：笔记标题、所属目录、最后修改时间

**笔记导入**：
- 支持复选框多选笔记，支持"全选当前页"
- 点击"导入"后批量导入选中笔记
- 导入过程中显示进度条（已完成/总数）
- 导入后格式转为 Markdown 作为新资料来源
- 导入完成后自动触发切片 + 向量化
- 重复导入同一笔记时提示"该笔记已导入，是否更新？"

**错误处理**：
- API Key 无效：提示"API Key 无效，请检查后重新输入"
- 网络超时：自动重试 3 次，失败后提示用户检查网络
- 笔记内容为空：跳过该笔记并记录日志，不影响其他笔记导入

**US-1.5 资料来源管理**
> 作为用户，我希望查看、搜索、重命名和删除已导入的资料来源。

验收标准：
- 列表展示：名称、类型图标（文件/URL/音频）
- 支持按名称实时搜索过滤
- 支持重命名和删除（删除需二次确认）
- 支持复选框多选
- 选中的来源用于限定 AI 检索范围
- 点击来源可查看完整的解析后 Markdown 内容和原内容

**US-1.6 网络搜索导入**
> 作为用户，我希望通过输入关键词搜索网络内容，并将搜索结果一键导入为资料来源，以便快速获取最新的网络资料。

验收标准：

**搜索功能**：
- 左栏导入入口提供"网络搜索"对话输入框
- 支持输入关键词进行搜索，支持直接填入对应网址导入（网址导入Tool）
- 搜索结果以列表形式展示，每条结果包含：标题、URL
- 搜索结果按相关度排序（综合考虑关键词匹配度、权威性、时效性）
- 显示 10 条最匹配的结果
- 搜索过程中显示加载状态


**导入功能**：
- 每条搜索结果后提供复选框，支持多选
- 支持"全选当前页"快捷操作
- 点击"导入选中"按钮，批量导入选中的网页
- 导入时复用已有的 URL 导入能力（MarkItDown MCP 抓取正文并转为 Markdown）
- 重复导入同一 URL 时提示"该网页已导入，是否更新？"
- 单次导入上限：最多 10 个网页

**预览功能**：
- 点击搜索结果标题可在新标签页打开原网页
- 导入后可在资料来源列表中查看完整的 Markdown 内容

**错误处理**：
- 网页抓取失败：跳过该网页并记录日志，不影响其他网页导入（复用 US-1.2 的错误处理逻辑）
- 搜索 API 配额耗尽：提示"搜索次数已达上限，请稍后再试"


#### 模块二：AI 对话

**US-2.1 基于资料的问答**
> 作为用户，我希望选中资料来源后提问，AI 基于这些资料内容回答问题。

验收标准：
- 如果用户不选任何资料来源的话就提示“请添加对应资料”
- 仅从用户当前选中的资料来源中检索（不跨笔记本和对话）
- 支持流式输出，回答实时显示
- 检索采用向量语义检索，按相关度排序

**US-2.2 对话管理**
> 作为用户，我希望管理多个对话，方便不同主题的讨论。

验收标准：
- 支持新建对话、切换对话、删除对话
- 对话列表按更新时间降序排列
- 默认以首条用户消息前 20 个字符作为标题，支持手动修改
- 删除对话需二次确认
- 对话历史自动保存到数据库

**US-2.3 保存对话为笔记**
> 作为用户，我希望将有价值的对话内容保存为笔记。

验收标准：
- 提供"保存为笔记"按钮
- 将当前轮次的问答内容打包为 Markdown 格式
- 保存为 Note（type: note），出现在笔记列表中
- 不自动转为资料来源

#### 模块三：内容生成

**US-3.1 一键生成（UI 按钮）**
> 作为用户，我希望选中资料后点击按钮，一键生成思维导图/PPT/测验/笔记。

验收标准：
- 右栏上方提供四个功能图标：思维导图、PPT、测验、笔记
- 未选中来源时图标置灰，hover 提示"请先选择资料来源"
- 点击后调用后端生成服务，结果保存为 Note 并显示在笔记列表
- 支持多种生成类型：
    - 🗺️ **思维导图**：Markmap 格式 Markdown，支持可视化渲染和源码切换
    - 📊 **PPT**：HTML 幻灯片，支持 iframe 预览和全屏演示
    - ❓ **测验**：多选题，默认 10 道，支持选择数量（5/10/15/20）
    - 📝 **笔记**：结构化 Markdown 摘要

**US-3.2 对话触发生成**
> 作为用户，我希望通过自然语言告诉 AI 生成思维导图或测验等。

验收标准：
- 用户说"帮我生成思维导图"等自然语言指令
- AI 识别意图后自动调用对应的生成服务
- 生成结果与 UI 按钮触发的结果完全一致
- AI 在对话中给出反馈（如"已为你生成思维导图"）

**US-3.3 思维导图交互**
> 作为用户，我希望生成的思维导图可以可视化查看和导出。

验收标准：
- 点击笔记 → 右栏内联展开渲染效果
- 支持"可视化 / Markdown 源码"切换视图
- 支持复制 Markdown 文本
- 支持导出为 HTML 文件

**US-3.4 PPT 交互**
> 作为用户，我希望生成的 PPT 可以预览和导出。

验收标准：
- 点击笔记 → 右栏内联展开，使用 iframe 预览
- 支持全屏演示模式
- 支持导出为 pptx 文件下载

**US-3.5 测验交互**
> 作为用户，我希望生成的测验可以逐题作答并查看解析。

验收标准：
- 以卡片形式逐题展示
- 用户选择答案后即时判定对错并展示解释
- 正确答案高亮，错误选项标红
- 支持"重新作答"按钮

#### 模块四：笔记管理

**US-4.1 笔记列表**
> 作为用户，我希望查看和管理所有生成的笔记。

验收标准：
- 按更新时间降序排列
- 支持按标题搜索
- 支持重命名、删除（需二次确认）、下载为 .md 文件
- 支持 Markdown 编辑器查看和编辑

**US-4.2 笔记转资料来源**
> 作为用户，我希望将笔记添加为资料来源，使其可被 AI 检索。

验收标准：
- 提供"添加到资料来源"操作
- 系统复制笔记内容创建新 Source，执行切片 + 向量化
- 笔记本身不变，标记 `is_source = true`
- 已添加为来源的笔记显示标记
- 删除笔记或来源互不影响




###  5. 账户体系

#### 5.1 注册

**字段**：邮箱、密码、邮箱验证码

**流程**：
1. 输入邮箱 → 点击"获取验证码"
2. 系统发送 6 位数字验证码（有效期 5 分钟）
3. 填写验证码和密码（8-20 位，至少包含字母和数字）
4. 校验通过 → 注册成功 → 跳转登录页

**约束**：
- 邮箱不可重复注册
- 验证码 60 秒内不可重复发送
- 验证码输入错误 5 次后锁定，需重新获取

#### 5.2 登录

**字段**：邮箱、密码

**流程**：
1. 输入邮箱和密码
2. 弹出人机验证（滑块/点选）
3. 验证通过 → 登录成功 → 进入主页

**约束**：
- 连续输错密码 3 次，锁定账户 15 分钟
- JWT Token 有效期 7 天

#### 5.3 找回密码

输入邮箱 → 获取验证码 → 填写验证码和新密码 → 密码重置成功 → 跳转登录页

### 7. 设置模块

#### 7.1 AI 服务配置

- 可添加多组 AI 配置，每组包含：API_KEY、Model、URL
- 每组配置可独立启用/禁用
- 配置保存后立即生效，无需重启
- 新增配置时根据模型名称自动填充上下文窗口大小
- 保存前自动验证 API_KEY 有效性

#### 7.2 路由与降级
- 只针对可以写一个接口就可以多组配置的服务
- 调用时按配置列表顺序尝试，首个成功的配置即为当前使用的配置
- 调用失败（超时、认证错误、限流等）时自动尝试下一个启用的配置
- 所有配置均失败时返回明确错误提示
- 不做后台健康探测，依赖调用时的实际结果判断可用性

#### 7.3 搜索 API 配置

- 每组配置包含：API_KEY、每日配额限制
- 配置保存后立即生效，无需重启
- 保存前自动验证 API_KEY 有效性
- 支持查看当前配额使用情况（已用/总量）
- 配额耗尽时自动切换备用搜索引擎（如有配置）

### 8. 个人中心

- **头像**：支持上传自定义头像（jpg/png，≤ 2MB），提供裁剪功能
- **邮箱**：仅展示，不可修改
- **修改密码**：需验证当前密码，修改成功后需重新登录





### 9. 降级策略

系统采用**渐进式降级**设计。对话、向量化、网页抓取等核心能力由服务端内置提供，用户无需配置即可使用。其余需要用户自行配置付费 API 的服务，均支持降级兜底，确保未配置时用户仍能使用基础功能。

#### 9.1 服务分级

| 级别 | 服务 | 提供方 | 用户是否需要配置 | 降级方式          |
|------|------|:---:|:---:|---------------|
| **P0 - 核心** | AI 对话（LLM） | 服务端 | 否 | 服务端内置，开箱即用    |
| **P0 - 核心** | 向量化服务（Embedding） | 服务端 | 否 | 服务端内置，开箱即用    |
| **P0 - 核心** | 网页抓取（MarkItDown MCP） | 服务端 | 否 | 服务端内置，开箱即用    |
| **P1 - 可选** | 语音转文本（ASR） | 用户配置 | 是 | 降级为免费的语音转文本模型 |
| **P1 - 可选** | 网络搜索 API | 用户配置 | 是 | 降级为简单的搜索导入功能  |
| **P2 - 增强** | PPT 生成服务 | 用户配置 | 是 | 降级为简单的ppt生成实现 |


#### 9.3 配置状态管理

- 设置页统一展示用户可配置服务的状态（已配置/未配置/配置失效）
- 未配置的服务在 UI 上有明确的禁用态和引导提示
- 支持"一键检测"按钮，批量验证所有已配置服务的可用性
- 配置变更后立即生效，无需重启
- 首次使用时弹出引导向导，标注哪些是可选增强配置

#### 9.4 降级通知

- 功能入口因未配置而降级时，显示对应的配置引导文案
- 设置页顶部汇总显示"当前有 N 项可选服务未配置"，点击可查看详情


###  非目标（Non-Goals）

以下功能**不在 V1 范围内**，明确排除以保护开发周期：
- 思维导图，测验等输出内容可以自定义提示词
- 对话历史的向量化检索（V1 仅对资料来源做向量化）
- 递归爬取网站子页面
- 非中文音频的语音识别
- 移动端适配（V1 仅桌面端，最小 1280px 宽度）
